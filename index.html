Tab

Micropayments for AI agents. Fiat-native. Sub-cent transactions.

The Problem

AI agents need to pay each other for services — API calls, data lookups, compute — but:

  • Crypto solutions (x402, Lightning) add complexity and require on-ramps
  • Traditional rails (Stripe) have per-transaction fees ($0.30+) making micropayments impossible
  • No pure-fiat A2A solution exists for sub-cent transactions

The Model

Tab works like Venmo — for agents:

  • Human does KYC once, funds account
  • Agents transact internally (free, instant, sub-cent)
  • Real money only moves on funding/withdrawal
Human funds $100 → Tab ledger
  ├── Agent A: balance
  ├── Agent B: balance
  └── Agent C: balance

Agent A pays Agent B $0.000003 for a lookup
→ Just a ledger entry, instant, free
→ Real money only moves on funding/withdrawal

For Humans

Your agent needs Tab to pay for tools and services.

  1. Sign up (coming soon)
  2. Complete identity verification (~2 minutes)
  3. Fund your account
  4. Give your agent the API key

For Tool Builders

Accept Tab payments in your MCP server:

from tab_payments import verify_payment

@server.tool()
@requires_payment(amount_microcents=2000)
async def lookup(query: str) -> str:
    """Look up company data."""
    return await do_lookup(query)
